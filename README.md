# Oasis Infobyte Data Analytics Internship Portfolio (OIBSIP)

## 📌 Overview
This repository showcases a comprehensive collection of data analytics and machine learning projects completed during my internship at **Oasis Infobyte**. The curriculum is divided into two levels, ranging from Exploratory Data Analysis (EDA) to advanced predictive modeling using supervised and unsupervised learning.

## 📊 Project Index

### **Level 01: Exploratory Data Analysis & Visualization**
| Project | Title | Description | Key Tech |
| :--- | :--- | :--- | :--- |
| **01** | [Retail Sales Analysis](./Level%2001/Project%2001) | Analyzing transaction patterns and demographics. | Pandas, Seaborn, Matplotlib |
| **02** | [Customer Segmentation](./Level%2001/Project%2002) | Segmenting customers using Silhouette scores. | K-Means, Silhouette Score |
| **03** | [NYC Airbnb Analysis](./Level%2001/Project%2003) | Geospatial and pricing analysis of rental data. | Geospatial Viz, Pandas |

### **Level 02: Advanced Analytics & Machine Learning**
| Project | Title | Description | Key Tech |
| :--- | :--- | :--- | :--- |
| **01** | [House Price Prediction](./Level%2002/Project%2001) | Predicting property values with Multiple Linear Regression. | RFE, VIF, Scikit-learn |
| **02** | [Wine Quality Prediction](./Level%2002/Project%2002) | Classifying wine quality based on chemical properties. | Random Forest, SVC, SGD |
| **03** | [Credit Card Fraud Detection](./Level%2002/Project%2003) | Detecting fraudulent transactions in imbalanced data. | SMOTE, Random Forest |

## 🛠️ Technical Skill Set
* **Data Preprocessing:** Feature Scaling (StandardScaler, Min-Max), Handling Class Imbalance (SMOTE), and Outlier Detection (IQR Method).
* **Machine Learning:** Linear Regression, Random Forest Classifier, Stochastic Gradient Descent (SGD), and Support Vector Machines (SVM).
* **Feature Selection:** Recursive Feature Elimination (RFE) and Multicollinearity checks using Variance Inflation Factor (VIF).
* **Evaluation Metrics:** Confusion Matrices, Precision-Recall Curves, and Silhouette Analysis.

## 🚀 Key Learning Objectives
* Developing end-to-end data pipelines from raw CSV ingestion to model deployment.
* Communicating complex data trends through heatmaps, boxplots, and time-series visualizations.
* Applying statistical validation to ensure model reliability and accuracy.
