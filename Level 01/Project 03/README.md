# NYC Airbnb Market Analysis — Project 03

## 📌 Project Overview
This project involves a comprehensive Exploratory Data Analysis (EDA) of the New York City Airbnb market using the 2019 dataset. The analysis focuses on understanding pricing dynamics, geographical distribution of listings, and host behaviors across the five boroughs to identify trends in the short-term rental market.

## 📊 Project Summary
| Feature | Details |
| :--- | :--- |
| **Dataset** | `AB_NYC_2019.csv` |
| **Objective** | Market Pricing & Geospatial Analysis |
| **Main Tools** | Python (Pandas, Matplotlib, Seaborn) |
| **Key Insight** | Price variance between Manhattan and other boroughs |

## 🚀 Analysis Workflow
1. **Data Cleaning:** Handled missing values in `reviews_per_month` and `host_name`. Cleaned data types for date-based analysis.
2. **Geospatial Analysis:** Visualized listing density across Neighborhood Groups (Manhattan, Brooklyn, Queens, Bronx, Staten Island).
3. **Price Distribution:** Analyzed price variations across different `room_type` categories (Entire home/apt, Private room, Shared room).
4. **Demand Metrics:** Investigated listing popularity based on the `number_of_reviews` and `availability_365`.
5. **Correlation Study:** Analyzed the relationship between location (latitude/longitude) and price points.

## 🛠️ Tech Stack
* **Data Manipulation:** `pandas`, `numpy`
* **Data Visualization:** `matplotlib`, `seaborn`
* **Environment:** Jupyter Notebook

## 📂 Project Structure
* `Project 03.ipynb` - The full notebook containing cleaning, EDA, and maps.
* `AB_NYC_2019.csv` - The New York City Airbnb open dataset.
